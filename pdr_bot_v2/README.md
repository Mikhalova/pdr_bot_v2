# 🚗 PDR Bot — Підготовка до іспиту ПДР

Telegram-бот для підготовки до іспиту з правил дорожнього руху України.  
Питання парсяться з сайту **pdrtest.com** за допомогою Playwright.

---

## 📋 Функції

| Режим | Опис |
|-------|------|
| 📋 Білети | 40 білетів по 20 питань |
| 📚 Теми | Питання за темами |
| 🎲 Випадкові | 20 випадкових питань |
| ❌ Помилки | Тільки питання де помилявся |
| ⏱ Іспит | 20 питань, режим іспиту |
| 📊 Статистика | Прогрес, серія правильних |

## 🚀 Локальний запуск

### Крок 1 — Клонуй і встанови залежності

```bash
git clone <repo>
cd pdr_bot
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Крок 2 — Отримай токен бота

1. Відкрий [@BotFather](https://t.me/BotFather) у Telegram
2. `/newbot` → введи назву і username
3. Скопіюй токен

### Крок 3 — Змінні середовища

```bash
export BOT_TOKEN="твій_токен_тут"
```

Або створи файл `.env`:
```
BOT_TOKEN=твій_токен_тут
```

### Крок 4 — Запарси питання

```bash
pip install -r requirements_scraper.txt
playwright install chromium
python scraper.py
```

Скрипт збереже `questions.json` у поточну папку.

### Крок 5 — Запусти бота

```bash
python bot.py
```

## ☁️ Деплой на Railway

1. Завантаж проект на GitHub
2. Відкрий [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub**
3. Вибери репозиторій
4. У **Variables** додай: `BOT_TOKEN = твій_токен`
5. Завантаж `questions.json` у репозиторій (після локального парсингу)
6. Railway автоматично запустить `worker: python bot.py` (з Procfile)

## 💡 Особливості реалізації

- **Варіанти відповідей** — відображаються як нумерований список у тексті питання (1/2/3/4), а не як довгі кнопки. Кнопки відповіді — тільки цифри.
- **Історія чату** — повідомлення НЕ видаляються, зберігається повна історія питань і відповідей.
- **Мова** — тільки українська.
- **Зображення** — питання зі знаками відправляються з фото.
- **Статистика** — зберігається список неправильних відповідей для режиму "Робота над помилками".
